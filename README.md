University Course Aggregator and Recommendation System
A Python application designed to collect, process, and display online course metadata from various university platforms, featuring a custom recommendation engine tailored to user preferences.

Features
Data Collection: Utilizes API integrations and web scraping via BeautifulSoup to extract course information.

Data Pipeline: Filters, cleans, and normalizes unstructured data using Pandas and Regular Expressions (Regex).

Graphs: Specificaly Designed Graphs to showcase the Correlation between Prices, Difficulty and Duration in a Course.

Recommendation Engine: Implements an algorithm that analyzes user preferences to sort and recommend relevant academic courses.

Graphical User Interface: Built a desktop UI using Tkinter with widgets and data visualizations via Matplotlib.

Tech Stack and Libraries
Language: Python
GUI: Tkinter
Data and Analytics: Pandas, Matplotlib, Regex
Web Scraping: BeautifulSoup
System: OS
